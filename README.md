Nova Commerce
Modern E-Commerce Frontend | Built with Next.js 14 (App Router)

Project Overview
Nova Commerce is a fully responsive e-commerce frontend application built using Next.js (App Router) and TypeScript. This project demonstrates clean architecture, modern UI design principles, dynamic routing, and state management using React Context API. It is structured as a production-ready frontend suitable for real-world deployment.

Key Features
	•	Product Listing Page with Pagination
	•	Search Functionality (Debounced API calls)
	•	Category Filtering
	•	Cart Management (Add, Remove, Quantity Update)
	•	Checkout Flow with Confirmation Modal
	•	Dynamic Order Details Page (/order/[id])
	•	Responsive Design (Mobile, Tablet, Desktop)
	•	Loading Skeletons for Enhanced UX
	•	Modern Premium UI Styling

Tech Stack
	•	Next.js 14 (App Router)
	•	TypeScript
	•	React Context API
	•	CSS (Custom Premium Styling)
	•	Vercel (Deployment)

Setup Instructions
	•	Clone the repository: git clone <repository-url>
	•	Navigate to the project folder: cd ecommerce-frontend
	•	Install dependencies: npm install
	•	Run the development server: npm run dev
	•	Open http://localhost:3000 in your browser
